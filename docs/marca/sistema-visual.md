# Sistema visual

La marca se reconoce por cuatro decisiones (Manual Interno de Marca v1.0):

## ENSENADA (la palabra)
La palabra es protagonista. Debe tener más fuerza que cualquier símbolo.

## Fotografía
La imagen cuenta primero: gente real, río, puerto, historia y cultura.

## Color
Cada línea de contenido usa un color para ordenar temas y energía — ver [`colores-y-tipografia.md`](colores-y-tipografia.md).

## Aire
Menos elementos, más impacto y lectura rápida.

## Qué evitar

Tono de folleto turístico, exceso de texto, iconos obvios, tono de oficina pública. Ver también [`sponsors-y-reglas.md`](sponsors-y-reglas.md).
